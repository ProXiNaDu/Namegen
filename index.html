Имя
    NameGen by Proxinadu

      Документация

      Шаблон

      В шаблон будут подставляться различные слова. При написании шаблона можно использовать любые символы, кроме символа доллара($). Эти символы при генерации имени никак не изменятся, что позволяет всячески кастомизировать имя (К примеру, можно добавить 'The' перед именем). Чтобы подставить в имя случайное слово, нужно в шаблоне оставить специальную метку. Метки начинаются с символа доллара($), после которого идет какой-либо символ. Вместо этой пары будет подставлено какое-либо слово из списка подстановок. Также между символом и долларом($) может стоять какое-либо число от 0 до 99, которое будет обозначать шанс появления подстановки этого рода в процентах (К примеру, если написать '$42s', то шанс того, что на это место что-то подставится будет равен 42%).

      Символы

      Символ - уникальный идентификатор для каждого списка подстановок. Желательно в качестве идентификатора выбирать именно 1 символ, хотя строка также будет работать. Генератор будет искать в шаблоне связки доллара($) и символа, после чего вместо этой связки будет подставлено одно из слов, связанного с заданным символом.

      Подстановки

      Подстановка - список слов, которые будут помещены вместо связки доллара($) и символа. Слова в списке должны быть разделены запятыми. Любые пробелы будут проигнорированы, поэтому их использование в словах не рекомендуется.
